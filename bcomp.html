¿Qué es Bootstrap?

Bootstrap es un framework de CSS desarrollado originalmente por Twitter que proporciona:

  • Componentes predefinidos listos para usar
  • Sistema de grillas responsivo
  • Clases utilitarias para estilizado rápido
  • JavaScript interactivo integrado
  • Compatibilidad cross-browser
Bootstrap 5

La versión más reciente y potente

¿Qué problemas resuelve?

Problemas Antes
  • Desarrollo lento
  • Inconsistencias de diseño
  • Problemas de responsividad
  • Incompatibilidad entre navegadores
  • CSS repetitivo
Soluciones Con Bootstrap
  • Desarrollo rápido
  • Diseño consistente
  • Responsivo por defecto
  • Compatible con todos los navegadores
  • Componentes reutilizables

Componente: Cards

¿Qué son las Cards?

Las cards son contenedores flexibles que pueden contener una variedad de contenido y opciones.

Estructura básica:

<div class="card">
  <div class="card-header">Header</div>
  <div class="card-body">
    <h5 class="card-title">Título</h5>
    <p class="card-text">Contenido</p>
  </div>
  <div class="card-footer">Footer</div>
</div>

Componente: Modal

¿Qué es un Modal?

Los modales son ventanas emergentes que se superponen al contenido principal.

Usos comunes:

  • Mostrar información adicional
  • Formularios de edición
  • Confirmaciones
  • Galerías de imágenes

Estructura básica:

<div class="modal fade" id="myModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título</h5>
        <button type="button" 
                class="btn-close" 
                data-bs-dismiss="modal">
        </button>
      </div>
      <div class="modal-body">
        Contenido del modal...
      </div>
      <div class="modal-footer">
        <button class="btn btn-primary">
          Guardar
        </button>
      </div>
    </div>
  </div>
</div>

Componente: Offcanvas

¿Qué es Offcanvas?

Un componente que permite mostrar contenido oculto desde los bordes de la pantalla.

Ideal para:

  • Menús de navegación
  • Carritos de compra
  • Paneles de filtros
  • Configuraciones

Código básico:

<!-- Botón trigger -->
<button class="btn btn-primary" 
        type="button" 
        data-bs-toggle="offcanvas" 
        data-bs-target="#offcanvasExample">
  Abrir Offcanvas
</button>

<!-- Offcanvas -->
<div class="offcanvas offcanvas-start" 
     id="offcanvasExample">
  <div class="offcanvas-header">
    <h5 class="offcanvas-title">
      Título
    </h5>
    <button type="button" 
            class="btn-close" 
            data-bs-dismiss="offcanvas">
    </button>
  </div>
  <div class="offcanvas-body">
    Contenido aquí...
  </div>
</div>

Componente: Carousel

¿Qué es un Carousel?

Un carrusel permite mostrar múltiples contenidos (imágenes, texto, etc.) en un espacio rotativo.

Ideal para:

  • Galerías de imágenes
  • Banners promocionales
  • Testimonios de clientes
  • Noticias destacadas

Partes principales:

  • Indicadores: Puntos para navegación directa
  • Slides: Contenido de cada diapositiva
  • Controles: Flechas anterior/siguiente

Características:

  • Rotación automática
  • Navegación manual
  • Indicadores de posición
  • Responsive por defecto
  • Altamente personalizable

Código completo:

<div id="myCarousel" class="carousel slide" 
     data-bs-ride="carousel" 
     data-bs-interval="2000">
  
  <!-- Indicadores -->
  <div class="carousel-indicators">
    <button data-bs-target="#myCarousel" 
            data-bs-slide-to="0" class="active"></button>
    <button data-bs-target="#myCarousel" 
            data-bs-slide-to="1"></button>
    <button data-bs-target="#myCarousel" 
            data-bs-slide-to="2"></button>
  </div>

  <!-- Slides -->
  <div class="carousel-inner">
    <div class="carousel-item active">
      <!-- Contenido slide 1 -->
    </div>
    <div class="carousel-item">
      <!-- Contenido slide 2 -->
    </div>
    <div class="carousel-item">
      <!-- Contenido slide 3 -->
    </div>
  </div>
  
  <!-- Controles -->
  <button class="carousel-control-prev" 
          data-bs-target="#myCarousel" 
          data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </button>
  <button class="carousel-control-next" 
          data-bs-target="#myCarousel" 
          data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>

<!-- Opciones importantes -->
<!-- data-bs-interval="5000" (velocidad) -->
<!-- data-bs-pause="hover" (pausar al hover) -->
<!-- data-bs-wrap="false" (no repetir) -->

Font Awesome

¿Qué es Font Awesome?

Es la biblioteca de iconos más popular del mundo, con más de 7,000 iconos gratuitos.

Ventajas:

  • Iconos vectoriales escalables
  • Fácil personalización con CSS
  • Completamente responsive
  • Carga rápida y eficiente

Instalación:

<!-- CDN -->
<link rel="stylesheet" 
      href="https://cdnjs.cloudflare.com/
           ajax/libs/font-awesome/6.4.0/
           css/all.min.css">

<!-- Uso básico -->
<i class="fas fa-star"></i>
<i class="far fa-heart"></i>
<i class="fab fa-github"></i>

Categorías:

fas - Solid far - Regular fab - Brands
