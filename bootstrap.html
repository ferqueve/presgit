¿Qué es Bootstrap?

Definición

Bootstrap es un framework front-end gratuito y de código abierto que facilita el desarrollo de sitios web responsive y mobile-first.

Historia

Creado por Mark Otto y Jacob Thornton en Twitter en 2010. Originalmente llamado "Twitter Blueprint".

Popularidad

Es el framework CSS más popular en GitHub con más de 160,000 estrellas y utilizado por millones de desarrolladores.

Sistema de Grillas

El sistema de grillas de Bootstrap está basado en 12 columnas y utiliza Flexbox.

Conceptos Clave:

  • Container: Envuelve el contenido y proporciona padding
  • Row: Agrupa las columnas horizontalmente
  • Column: Define el ancho del contenido

Breakpoints y Tamaños de Pantalla

Bootstrap define 6 breakpoints para diferentes tamaños de dispositivos:

Breakpoint Prefijo de Clase Ancho Mínimo Dispositivo
Extra Small col- < 576px Teléfonos
Small col-sm- ≥ 576px Teléfonos grandes
Medium col-md- ≥ 768px Tablets
Large col-lg- ≥ 992px Laptops
Extra Large col-xl- ≥ 1200px Desktops
Extra Extra Large col-xxl- ≥ 1400px Pantallas grandes
