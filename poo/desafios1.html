Desafío 3

Código original en Python

on = True
print("Encendido: ", on)

Consigna

El siguiente código Python asigna un valor a una variable llamada on

que represente el estado de un dispositivo que puede estar encendido o no; luego

imprime el valor de esa variable on en la consola:

Escribe a continuación el código equivalente en C#. Ten en cuenta que:

  • En C# vas a tener que declarar el tipo de la variable on, que será bool.
  • El valor True en Python es true en C# y el de False es false.
  • Console.WriteLine() en C# no permite múltiples argumentos separados por

comas, a diferencia de print() en Python; vas a tener que usar "Encendido:

" + on para lograr que se imprima en la consola lo mismo que en Python.

Desafío 7

Código original en Python

nombre = "Juan"
edad = 30
print(f"Hola, me llamo ", nombre, " y tengo ", edad, " años.")

Consigna

El código Python a continuación permite imprimir en la consola una cadena de

caracteres embebiendo expresiones, en particular, variables. La cadena "Hola,

me llamo Juan y tengo 30 años" se obtiene como resultado de asignar nombre =

"Juan" y edad = 30 y luego concatenar "Hola, me llamo " con nombre, con

" y tengo ", con edad y con " años.".

Desafío 9

Código original en Python

print(text[0])
print(text[-1])

Consigna

El siguiente código Python asigna una cadena de texto a una variable llamada

text; luego imprime el primer y el último carácter de esa variable text en

la consola:

Escribe a continuación el código equivalente en C#. Ten en cuenta que el índice

del último caracter en C# es text.Length - 1, porque no puede haber un índice

negativo a diferencia de Pyhton. text.Length obtiene la cantidad de caracteres

de la cadena y text.Length - 1 es el índice del último porque el índice del

primero es 0.
