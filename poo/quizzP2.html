Bienvenido a C#

Guía de transición desde Python y C++ para dominar .NET

La Clase

Es el "molde" o plano. En C#, a diferencia de Python, todo debe estar dentro de una clase y los archivos deben coincidir con el nombre de la clase pública.

// C# public class Persona { // Cuerpo }

Atributos (Campos)

Son las variables dentro de la clase. En C# usamos modificadores de acceso (public, private) y tipos de datos estrictos (int, string).

// C# public string Nombre; private int _edad; // Python: self.nombre // C++: this->nombre

Mensajes (Métodos)

Son las acciones que el objeto puede realizar. Se definen con un tipo de retorno (o void) y parámetros tipados.

// C# public void Saludar() { Console.WriteLine("Hola"); }

Diferencias Clave

  • Tipado Estático: Debes declarar el tipo de variable (int x = 5;) a diferencia de Python.
  • Punto y Coma: Cada instrucción termina con ; como en C++.
  • Main: El punto de entrada es un método static void Main(string[] args).
  • Console: Para imprimir usamos Console.WriteLine().

Creando tu primer proyecto en Rider

1
Abrir Rider y Crear Nuevo Proyecto

Al iniciar, selecciona "Create New Project" o ve a File > New Project.

2
Seleccionar la Plantilla

Busca en la barra lateral izquierda .NET y selecciona Console App. Asegúrate de que diga "C#" y no "F#" o "VB".

3
Configurar Nombre y Ubicación

Ponle nombre a tu proyecto (ej: MiPrimerCSharp). Rider creará una carpeta con ese nombre y un archivo Program.cs.

4
Entendiendo Program.cs

Verás algo así:

namespace MiPrimerCSharp; // Espacio de nombres (como carpetas lógicas) class Program // Clase principal { static void Main(string[] args) // Punto de entrada { Console.WriteLine("Hello, World!"); } }
5
Ejecutar

Presiona el botón de "Run" en la parte superior o usa Ctrl+F5 (Windows) / Cmd+F5 (Mac).
