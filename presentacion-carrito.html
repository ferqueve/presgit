Fetch API - Obtener Productos

Código para cargar productos desde una API:
async function cargarProductos() {
  try {
    const response = await fetch('api/productos');
    
    if (!response.ok) {
      throw new Error('Error al cargar productos');
    }
    
    const productos = await response.json();
    mostrarProductos(productos);
    
  } catch (error) {
    console.error('Error:', error);
    Swal.fire('Error', 
      'No se pudieron cargar los productos', 
      'error');
  }
}
Conceptos clave:
  • async/await para manejo asíncrono limpio
  • fetch() realiza la petición HTTP GET
  • response.json() convierte la respuesta a objeto JavaScript
  • try/catch maneja errores de red o servidor
  • SweetAlert muestra mensajes de error elegantes

Arrays - Métodos para Agregar Productos

push() Agrega al final del array
let carrito = [];

function agregarAlCarrito(producto) {
  carrito.push({
    id: producto.id,
    nombre: producto.nombre,
    precio: producto.precio,
    cantidad: 1
  });
  actualizarCarrito();
  
  // Animación y alerta
  Swal.fire({
    icon: 'success',
    title: '¡Producto agregado!',
    timer: 1500
  });
}
unshift() Agrega al inicio del array
let carrito = [];

function agregarAlInicio(producto) {
  carrito.unshift({
    id: producto.id,
    nombre: producto.nombre,
    precio: producto.precio,
    cantidad: 1
  });
  actualizarCarrito();
}
Nota: push() es más común para carritos ya que muestra productos en el orden en que se agregaron.

Arrays - Métodos para Quitar Productos

splice() Elimina por índice (mutable)
function quitarProductoPorIndice(indice) {
  carrito.splice(indice, 1);
  actualizarCarrito();
}

// Ejemplo de uso
quitarProductoPorIndice(2); // Quita el 3er elemento
splice() modifica el array original directamente
filter() Elimina por condición (inmutable)
function quitarProductoPorId(productoId) {
  carrito = carrito.filter(
    item => item.id !== productoId
  );
  actualizarCarrito();
}

// Ejemplo de uso
quitarProductoPorId(42); // Quita producto con id 42
filter() crea un nuevo array. Es más funcional y previene errores.

Arrays - Buscar y Actualizar Productos

find() Buscar un producto
const producto = carrito.find(
  item => item.id === productoId
);

if (producto) {
  console.log('Encontrado:', producto);
}
findIndex() Localizar posición
const indice = carrito.findIndex(
  item => item.id === productoId
);

if (indice !== -1) {
  console.log('Posición:', indice);
}
map() Actualizar cantidades (crea nuevo array)
function actualizarCantidad(productoId, nuevaCantidad) {
  carrito = carrito.map(item => {
    if (item.id === productoId) {
      return { ...item, cantidad: nuevaCantidad };
    }
    return item;
  });
  actualizarCarrito();
}

Cálculo de Subtotal con reduce()

El método reduce() es ideal para sumar totales:
function calcularSubtotal() {
  const subtotal = carrito.reduce((acumulador, item) => {
    return acumulador + (item.precio * item.cantidad);
  }, 0);
  
  return subtotal;
}

// Ejemplo de uso
const subtotal = calcularSubtotal();
console.log(`Subtotal: $${subtotal.toFixed(2)}`);
Cómo funciona reduce():
  • Acumulador: Guarda el total acumulado en cada iteración
  • Item: Producto actual del array que se está procesando
  • Valor inicial (0): Punto de partida del acumulador
  • Return: Suma precio × cantidad al acumulador
Ventaja: Convierte un array completo en un solo valor numérico de forma elegante y funcional.

Envío urgente: Cuesta el 3% del valor total del carrito.
JavaScript para calcular costo de envío:
function calcularCostoEnvio(subtotal) {
  const tipoEnvio = document
    .getElementById('tipoEnvio').value;
  
  if (tipoEnvio === 'urgente') {
    return subtotal * 0.03; // 3%
  }
  
  return 0; // Envío normal es gratis
}

// Ejemplo de uso
const subtotal = 1000;
const costoEnvio = calcularCostoEnvio(subtotal);
console.log(`Envío: $${costoEnvio}`); 
// $30 si es urgente, $0 si es normal

Cálculo del Total Final

Función completa para calcular el total:
function calcularTotal() {
  // 1. Calcular subtotal
  const subtotal = carrito.reduce((acc, item) => {
    return acc + (item.precio * item.cantidad);
  }, 0);
  
  // 2. Calcular costo de envío
  const costoEnvio = calcularCostoEnvio(subtotal);
  
  // 3. Calcular total
  const total = subtotal + costoEnvio;
  
  // 4. Actualizar el DOM
  document.getElementById('subtotal').textContent = 
    `$${subtotal.toFixed(2)}`;
  document.getElementById('envio').textContent = 
    `$${costoEnvio.toFixed(2)}`;
  document.getElementById('total').textContent = 
    `$${total.toFixed(2)}`;
  
  return total;
}
Proceso paso a paso:
  1. Calcular subtotal usando reduce() para sumar precio × cantidad de cada producto
  2. Calcular envío aplicando 3% si es urgente, 0 si es normal
  3. Sumar ambos valores para obtener el total final
  4. Actualizar interfaz mostrando los valores formateados en el DOM
toFixed(2) formatea números a 2 decimales para mostrar precios correctamente ($19.99)

Enviar Comentario con POST

async function enviarComentario(productoId, comentario, calificacion) {
  try {
    const response = await fetch('api/comentarios', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        productoId: productoId,
        texto: comentario,
        estrellas: calificacion,
        fecha: new Date().toISOString()
      })
    });
    
    if (!response.ok) {
      throw new Error('Error al enviar comentario');
    }
    
    const resultado = await response.json();
    
    // Mostrar mensaje de éxito con SweetAlert
    Swal.fire({
      icon: 'success',
      title: '¡Éxito!',
      text: 'Comentario enviado correctamente',
      confirmButtonColor: '#10B981'
    });
    
    // Limpiar formulario
    document.getElementById('formComentario').reset();
    
  } catch (error) {
    console.error('Error:', error);
    Swal.fire({
      icon: 'error',
      title: 'Error',
      text: 'No se pudo enviar el comentario'
    });
  }
}
Elementos POST:
  • method: 'POST' para crear recursos
  • headers especifica tipo de contenido JSON
  • body contiene los datos a enviar
  • JSON.stringify() convierte objeto a string JSON
Mejores prácticas:
  • Validar respuesta con response.ok
  • Mostrar feedback al usuario con SweetAlert
  • Limpiar formulario con reset()
  • Manejar errores gracefully

function quitarProductoConAnimacion(id, elemento) {
  // Animación de salida
  elemento.classList.add(
    'animate__animated', 
    'animate__fadeOutRight'
  );
  
  // Esperar a que termine la animación

Animaciones útiles:

  • fadeOutRight - Salida hacia derecha
  • bounceIn - Entrada con rebote
  • slideInLeft - Deslizar desde izq.
  • pulse - Pulso (llamar atención)
  • shake - Agitar (error)
  • zoomIn - Acercar con zoom
Cómo usar:
  1. Importar Animate.css en el HTML
  2. Agregar clase animate__animated
  3. Agregar clase de animación específica
  4. Usar animationend para ejecutar código al terminar

LocalStorage - Persistencia de Datos

Guardar y cargar el carrito:
// Guardar carrito en localStorage
function guardarCarrito() {
  localStorage.setItem(
    'carrito', 
    JSON.stringify(carrito)
  );
}

// Cargar carrito desde localStorage
function cargarCarrito() {
  const carritoGuardado = 
    localStorage.getItem('carrito');
  
  if (carritoGuardado) {
    carrito = JSON.parse(carritoGuardado);
    renderizarCarrito();
    actualizarTotal();
  }
}

// Limpiar carrito
function vaciarCarrito() {
  carrito = [];
  localStorage.removeItem('carrito');
  renderizarCarrito();
  actualizarTotal();
}

// Llamar al cargar la página
window.addEventListener('DOMContentLoaded', () => {
  cargarCarrito();
});
Métodos localStorage:
  • setItem(key, value) - Guardar dato
  • getItem(key) - Recuperar dato
  • removeItem(key) - Eliminar dato
  • clear() - Limpiar todo el storage
Importante:

localStorage solo guarda strings, por eso necesitamos:

  • JSON.stringify() para convertir objetos a string antes de guardar
  • JSON.parse() para convertir el string de vuelta a objeto al recuperar
Con localStorage, el carrito persiste incluso si el usuario cierra el navegador.

Mejores Prácticas de Desarrollo

Validación de datos

Siempre valida la entrada del usuario antes de procesar, tanto en cliente como en servidor.

Manejo de errores

Usa try/catch en operaciones asíncronas y proporciona mensajes claros al usuario.

Feedback al usuario

Informa de cada acción con alertas, notificaciones o cambios visuales en la interfaz.

Modularización

Separa funciones por responsabilidad: renderizar, calcular, validar, guardar.

Persistencia

Guarda el carrito en localStorage para no perder datos entre sesiones.

Accesibilidad

Usa etiquetas semánticas, atributos ARIA y asegura navegación por teclado.

Rendimiento

Evita manipulaciones excesivas del DOM, actualiza en lote cuando sea posible.

Seguridad

Valida datos en el servidor, sanitiza entrada, usa HTTPS para transacciones.
