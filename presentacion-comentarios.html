1. Sistema de Likes Simple

El sistema más básico: un botón que cambia de estado al hacer clic.

HTML:
<button id="likeBtn" onclick="toggleLike()">
  <i class="fas fa-heart"></i> 0
</button>

JavaScript:
let likes = 0;
let isLiked = false;

function toggleLike() {
  if (isLiked) {
    likes--;
  } else {
    likes++;
  }
  isLiked = !isLiked;
  document.getElementById('likeCount').textContent = likes;
}
¡Pruébalo!

2. Sistema de Calificación por Estrellas

Permite calificar de 1 a 5 estrellas.

HTML:
<div id="stars">
  <i class="fas fa-star" onclick="rate(1)"></i>
  <i class="fas fa-star" onclick="rate(2)"></i>
  <i class="fas fa-star" onclick="rate(3)"></i>
  <i class="fas fa-star" onclick="rate(4)"></i>
  <i class="fas fa-star" onclick="rate(5)"></i>
</div>

JavaScript:
function rate(stars) {
  const allStars = document.querySelectorAll('#stars i');
  allStars.forEach((star, index) => {
    if (index < stars) {
      star.classList.add('active');
    } else {
      star.classList.remove('active');
    }
  });
}

3. Sistema de Corazones (Like Animado)

Un like con animación y cambio de color.

CSS:
.heart { color: #ddd; }
.heart.liked { color: #e74c3c; }
.heart:hover { transform: scale(1.2); }

JavaScript:
function toggleHeart(element) {
  element.classList.toggle('liked');
  element.classList.add('animate__animated');
  element.classList.add('animate__heartBeat');
  setTimeout(() => {
    element.classList.remove('animate__heartBeat');
  }, 1000);
}

4. Sistema de Comentarios

Permite agregar comentarios a una lista.

HTML:
<input id="commentInput" placeholder="Escribe un comentario">
<button onclick="addComment()">Enviar</button>
<div id="commentsList"></div>

JavaScript:
function addComment() {
  const input = document.getElementById('commentInput');
  const text = input.value;
  if (text === '') return;

  const list = document.getElementById('commentsList');
  const comment = document.createElement('div');
  comment.className = 'comment-item';
  comment.textContent = text;
  list.appendChild(comment);
  input.value = '';
}

5. Comentarios con Usuario y Fecha

Comentarios más completos con información adicional.

JavaScript:
function addCommentFull() {
  const text = document.getElementById('input').value;
  const user = document.getElementById('user').value;
  const date = new Date().toLocaleString();

  const comment = `
    <div class="comment-item">
      <strong>${user}</strong> - ${date}
      <p>${text}</p>
    </div>
  `;
  list.innerHTML += comment;
}
